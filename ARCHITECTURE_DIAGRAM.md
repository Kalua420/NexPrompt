в”ңв”Җ HTTPS only (SSL/TLS)
в”ңв”Җ Separate domains (isolation)
в””в”Җ Nginx reverse proxy (no direct backend access)

Layer 2: Nginx
в”ңв”Җ Rate limiting
в”ңв”Җ Security headers
в”Ӯ  в”ңв”Җ X-Frame-Options: DENY (admin), SAMEORIGIN (user)
в”Ӯ  в”ңв”Җ X-Content-Type-Options: nosniff
в”Ӯ  в”ңв”Җ X-XSS-Protection: 1; mode=block

в”ңв”Җ JWT authentication (access + refresh tokens)
в”ңв”Җ Role-based access control
в”Ӯ  в”ңв”Җ Backend validates user.role === 'admin'
в”Ӯ  в””в”Җ Frontend checks role before rendering
в””в”Җ Cross-domain redirects for unauthorized access

Layer 4: Database
в”ңв”Җ Parameterized queries (Prisma ORM)
в”ңв”Җ Password hashing (bcrypt)
в””в”Җ Environment-based credentials

- **Separate builds** for optimal bundle sizes
- **Enhanced security** with stricter headers for admin app
